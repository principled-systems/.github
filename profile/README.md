The <a href="https://principled.systems">Principled Systems Group</a> at the <a href="https://www.mpi-sws.org">Max Planck Institute for Software Systems (MPI-SWS)</a> was led by <a href="https://andrea.lattuada.me">Andrea Lattuada</a>.
We designed, implemented, and verified systems software, construed broadly[^1] and focus on the advancement of verification tooling and methodology for systems verification.

You can find more information about the group at [https://andrea.lattuada.me/principled.systems/](https://andrea.lattuada.me/principled.systems/)

[^1]: Inspired by <a href="https://systopia.cs.ubc.ca">Systopia</a>'s definition of the scope of Systems research.
